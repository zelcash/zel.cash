ZelCore v3.11.0 changes

Features

  • Bitcoin Testnet is now available
  • Ethereum Ropsten Testnet is now available
  • Economy mode is now called Simple mode
  • Following ERC20 Tokens are available: Uniswap (UNI), Coin Artist Token (COIN)
  • Following TRC20 Token is now available: JUST (JST)
  • It is now possible to send to bech32 address on Bitcoin and Litecoin
  • It is now possible to send zero value transactions on Ethereum, Ethereum Classic and Ethereum Ropsten
  • RVN asset list updated
  • Offline storing of encrypted transactions - allows for faster loading and viwien entire wallet without internet connectivity
  • Added notifications for tokens that already migrated to custom blockchains
  • Coins based on UTXO are now showing sats/Byte in fee section when constructing transaction
  • Any ZelCore user can now import an ERC20 Ethereum Token (previously only ZelCore+ feature)
  • It is now possible to buy an asset on quick swap exchanges that is not activated in your active exchange wallet portfolio
  • Ontology network fees can now have custom Gas Limit and custom Gas Price
  • Check how many Zelnodes are running specific zelApp on Zelnodes App

Bug fixes

  • Fixes possible Tron NaN balances and wrongly showing fee on some transactions
  • There was an edge case of showing different WIF of private key other than a selected coin and wallet
  • Corrects UI positioning of Wallet Connect
  • Fixes deeplinking on Linux and macOS
  • Minor fixes

  • Genesis (GENX) is now available as TRC20 token

  • If ZelCore+ is active, panel now reflect that with ZelCore+ logo

  • Bitcore is now integrated as ElectrumX solution rather than Insight
